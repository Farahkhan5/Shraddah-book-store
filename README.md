# Shraddah-book-store

An online bookstore web application developed using ASP.NET MVC with Entity Framework and Code-First Approach (ORM).  
This project allows users to browse books, manage categories and products, register/login, and add books to the cart.

---

## Features

- User Registration & Login
- Home Page
- About Page
- Contact Page
- Category Management
- Product Management
- Add to Cart Functionality
- Entity Framework Integration
- Code-First Approach
- ORM-Based Database Handling

---

## Installation & Setup

1. Clone the repository
2. Open project in Visual Studio
3. Restore NuGet Packages
4. Run Update-Database
5. Press Ctrl + F5 to run the project
